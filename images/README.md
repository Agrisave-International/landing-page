## Image Optimization Guidelines

### Format
- Use WebP for photos with JPG fallback
- Use PNG for logos and graphics with transparency
- Use SVG for icons when possible

### Naming Convention
- Use descriptive names: `corn-farmer-iowa-harvest.webp`
- NOT: `IMG_1234.jpg`
- Use hyphens, not underscores
- All lowercase

### Compression
- Target: <200KB per image
- Use tools: TinyPNG, Squoosh, ImageOptim
- Maintain quality while reducing file size

### Dimensions
- Hero images: 1200-1600px wide
- Feature cards: 600-800px wide
- Thumbnails: 300-400px wide
- Always provide width/height attributes in HTML

### Alt Text Examples
- Good: "Iowa corn farmer viewing NDVI crop health map on smartphone using AGRISAVE satellite monitoring app"
- Bad: "Farmer with phone"
- Include context, action, and relevant keywords (satellite, NDVI, crop health, AI)

## Stock Photo Sources
- Unsplash (free, high quality)
- Pexels (free)
- iStock (paid, authentic farmer photos)
- Getty Images (paid, professional)

## Photography Tips
- Use real American farmers when possible
- Show diversity (age, gender, ethnicity, farm types)
- Authentic farm settings (not staged)
- Modern technology + traditional farming
- Natural lighting preferred
- Show actual product usage when possible
