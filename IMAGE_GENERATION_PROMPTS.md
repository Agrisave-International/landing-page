## Tips for Best Results

### General Guidelines:
1. **Use "photorealistic" or "professional photography"** for hero and testimonial images
2. **Specify "4K quality" or "high detail"** for better resolution
3. **Include "natural lighting"** for outdoor farm scenes
4. **Add "clean modern design"** for UI/dashboard screenshots
5. **Specify color palette:** "green and brown agricultural colors"

### For Midjourney:
- Add `--ar 16:9` for landscape images
- Add `--ar 1:1` for square images
- Add `--v 6` for latest version
- Add `--style raw` for more photorealistic results

### For DALL-E 3:
- Be specific about composition and lighting
- Mention "professional" and "high quality"
- Avoid copyrighted terms

### For Stable Diffusion:
- Use detailed prompts with specific camera angles
- Add negative prompts: "blurry, low quality, distorted"
- Use models trained on photography for realistic images

## Image Optimization After Generation

1. **Resize** to appropriate dimensions
2. **Convert** to WebP format (use cwebp or online tools)
3. **Compress** to reduce file size (aim for <200KB)
4. **Add** descriptive filenames (not IMG_1234.jpg)
5. **Test** on different devices and screen sizes

### Recommended Tools:
- **TinyPNG.com** - Easy compression
- **Squoosh.app** - Format conversion and compression
- **cwebp** - Command line WebP conversion
- **ImageOptim** (Mac) - Batch optimization
